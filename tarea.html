ELEMENTOS HTML

Elemento Descripcion Documentacion
html Representa a la raiz del documento HTML documentacion
div El elemento de división de contenido documentacion
header Representa un grupo de ayudas introductorias o de navegación documentacion
article Representa una composición auto-contenida en un documento documentacion
section Representa una sección genérica de un documento documentacion
nav Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos documentacion
aside Representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento documentacion
footer Representa un pie de página para el contenido de sección más cercano o el elemento raíz de sección documentacion
main Representa el contenido principal del body de un documento o aplicación documentacion
p Es el apropiado para distribuir el texto en párrafos documentacion
br Produce un salto de línea en el texto (retorno de carro) documentacion
hr Representa un cambio de tema entre párrafos documentacion
wbr Representa una posición dentro del texto donde el explorador puede opcionalmente saltar una línea , aunque sus reglas de salto de línea de otra manera no crearían un salto en esa posición documentacion
sup Define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice documentacion
sub Define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice documentacion
strong Es el apropiado para marcar con especial énfasis las partes más importantes de un texto documentacion
em Es el apropiado para marcar con énfasis las partes importantes de un texto documentacion
mark Representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o importancia en un contexto particular documentacion
q Indica que el texto adjunto es una cita corta en línea documentacion
cite Marca una referencia a una fuente, o el autor de un texto citado documentacion
blockquote Crea citas en bloque, marca las citas a otros autores o documentos documentacion
dfn Sirve para marcar el término que se quiere definir documentacion
abbr Representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura documentacion
small Hace el tamaño del texto una talla más pequeña (por ejemplo, de largo a mediano, o de pequeño a extra pequeño) que el tamaño mínimo de fuente del navegador documentacion
ins Marca las partes de un texto que han sido añadidos al documento documentacion
del Marca las partes de un texto o documento que han sido suprimidas o sustituidas documentacion
s Muestra el texto tachado con una linea horizontal documentacion
code Es el apropiado para marcar el código de un programa documentacion
samp Se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador documentacion
kbd Marca el texto que debe introducir el usuario documentacion
pre Representa texto preformateado documentacion
address Aporta información de contacto para su article más cercano o ancestro body documentacion
